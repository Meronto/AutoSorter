## Installation

1. Clone the repository: git clone "https://github.com/meronto/file-sorter.git"
cd file-sorter

2. Install dependencies: sudo apt install python3-tk gcc

3. Run the GUI: python3 gui/sorter-gui.py


## Usage

1. Select the source folder containing files
2. Select the destination folder for sorted files
3. Click "Start Sorting"
4. Files will be automatically sorted into categories

## Configuration

Edit `config/config.conf` to add new file extensions
